# Aru Reference

This is a compact reference for **Aru v1.0.0**.

## Core Sentence

```text
subject ra predicate
subject ra predicate ke object
```

Examples:

```text
na ra waro.
na ra sela ke luma.
```

## Core Particles

| Particle | Function |
|---|---|
| `ra` | predicate boundary |
| `ke` | direct object |
| `ne` | weak relation |
| `le` | context frame |
| `re` | relative phrase |
| `ri` | relative substitute |
| `no` | negation |
| `ka` | yes/no question |
| `o` | imperative |
| `ya` | name / foreign word |

## Pronouns

```text
na -> I / we
ta -> you
u  -> he / she / it / they
```

Explicit plurality:

```text
na mun
ta mun
u mun
mene mun
```

## Time

```text
pa -> past
nu -> now / present
po -> future
```

Time stands after `ra`:

```text
na ra pa tanu.
na ra po naka.
```

## Relations

```text
to -> area / place / contact
ko -> direction / addressee / goal
so -> source
we -> accompaniment / means
ne -> weak relation
```

Examples:

```text
na ra to malo.
na ra naka ko malo.
na ra naka so malo.
na ra naka we ta.
malo ne na
```

## Context

```text
X le, Y
```

Examples:

```text
temu luma nu le, na ra waro.
etu ta ra naka le, na ra naka.
malo ne na le, ta ra naka ko i.
```

## Content

```text
ke i le, sentence
```

Example:

```text
na ra sona ke i le, u ra pa tanu.
```

## Relative Phrases

```text
head re ... ri ...
```

Examples:

```text
mene re ri ra maku ke malo
malo re na ra sela ke ri
malo re na ra to ri
```

## Questions

```text
ka ta ra kani?
se ra naka?
ta ra sela ke se?
ta ra to se?
ta ra no tanu so etu se?
```

## Comparison

```text
X ra Q pasu Y
X ra Q sama pasu Y
X ra Q mini pasu Y
X ra Q pasu ora
```

Examples:

```text
malo i ra maru pasu malo ena.
malo i ra maru sama pasu malo ena.
wai i ra welo mini pasu wai ena.
malo i ra maru pasu ora.
```

## Style Rule

```text
Better two clear short phrases than one long unclear phrase.
```
